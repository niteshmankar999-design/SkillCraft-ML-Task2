# Customer Segmentation using K-Means Clustering

## SkillCraft Technology Internship - Task 2

### Project Overview
This project performs customer segmentation using the K-Means Clustering Machine Learning algorithm.

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

### Dataset
Mall Customers Dataset

### Machine Learning Algorithm
K-Means Clustering

### Features Used
- Annual Income
- Spending Score

### Objective
To group customers into different clusters based on spending behavior and annual income.

### Output
The model successfully segments customers into multiple groups and visualizes them using clustering graphs.
